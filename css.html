CSS Learning Notes

1. Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages.

It controls the colors, fonts, spacing, and overall look of your HTML content.

CSS uses selectors to target HTML elements and apply styles.

2. Basic CSS Syntax


selector {
  property: value;
}

Example:


p {
  color: blue;
  font-size: 16px;
}

3. How to Include CSS

There are three ways to add CSS to HTML:

  • Inline styles: <h1 style="color:red;">Hello</h1>
  • Internal stylesheet (inside <style> tag)
  • External stylesheet: <link rel="stylesheet" href="styles.css">

4. Selectors

Selectors specify which elements the CSS rules apply to.


/* Element selector */
h1 {
  color: green;
}

/* Class selector */
.my-class {
  background-color: yellow;
}

/* ID selector */
#unique {
  border: 1px solid black;
}

/* Attribute selector */
input[type="text"] {
  border: 2px solid blue;
}
/* universal selector */
*{
  padding:16px;
}

/* Descendant selector */
nav a {
  color: purple;
}

5. Background Color

The background-color property sets the background color of an element.


body {
  background-color: #f0f0f0;
}

header {
  background-color: #333;
  color: white;
  padding: 20px;
}

You can use color names, HEX, RGB, or HSL values.

6. Styling a Button

Buttons can be styled using CSS for better UI.


button {
  background-color: #007bff;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 5px;
  cursor: pointer;
  transition: background-color 0.3s ease; 
}

button:hover {
  background-color: #0056b3;
}

This adds color, padding, rounded corners, and a hover effect.

7. Navigation Bar Styling

Basic styling for a navigation bar:


nav {
  background-color: #222;
  padding: 10px;
}

nav ul {
  list-style: none;
  display: flex;
  gap: 15px;
}

nav ul li a {
  color: white;
  text-decoration: none;
  padding: 5px 10px;
}

nav ul li a:hover {
  background-color: #555;
  border-radius: 3px;
}

Use flexbox to arrange nav links horizontally.

8. Positioning

CSS positioning controls how elements are placed on the page.

  • static (default) - normal flow
  • relative - positioned relative to its normal position
  • absolute - positioned relative to nearest positioned ancestor
  • fixed - fixed relative to the viewport
  • sticky - toggles between relative and fixed based on scroll

.box {
  position: relative;
  top: 10px;
  left: 20px;
}

.popup {
  position: absolute;
  top: 50px;
  right: 10px;
}

.header {
  position: fixed;
  top: 0;
  width: 100%;
  background-color: #333;
  color: white;

}
