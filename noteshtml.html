HTML Learning Notes

1. Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages.

It uses tags like <p> or <h1> to structure content.

2. Basic HTML Structure


<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
  </body>
</html>

3. Headings, Paragraphs, and Links

Use <h1> to <h6> for headings.

Use <p> for paragraphs.</p>

Use <a href="url"> for links.</a>

Use <br> for breaking a text and it is self-closing tag.<br>

Use <u> for underlining a text.</u>

Use <b> for bolding a text.</b>

4. Lists, Images, and Tables

Unordered List:


<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Image: <img src="image.jpg" alt="Text">

Table:


<table>

  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>30</td></tr>
</table>

5. Forms

Forms collect user input.


<form>
  <label>Name:
  <input type="text" placeholder="Enter your name">
  </label>
  <br><br>
  <label>Email:
  <input type="email" placeholder="Enter your email">
  </label>
</form>

Common input types:

  • text - single line text
  • email - email address
  • password - password field
  • radio - select one option
  • checkbox - select multiple options
  • submit - submit button
